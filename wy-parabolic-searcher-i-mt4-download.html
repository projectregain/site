Assistant Indicator. Multiple Instruments. Multiple Timeframes. Multiple digits. It identifies different types of parabolas of “n” degree with the vertex at a distant candle and at a near candle. It uses the mean absolute deviation method for searching. It has multiple configurable parameters. Simple and convenient tool. It is a lite version of the product WY Pattern Searcher P6 I MT4.

### The indicator parameters

Main:

  • Show Basic Spline – display the main spline

    • Show Help Spline – display the auxiliary spline
    • Show Spline Labels – display the spline names (configured below)
    • Use Following Spline – enable the following mode (pattern are drawn as they form)
    • Play Warn. Sound On Detect – play a sound signal when a pattern is found
    • Play Warn. Sound On Following – play a sound signal when the current pattern is completely drawn
    • Play Warn. Sound On Following Break – play a sound signal when the current pattern changes
    • Clean Old Splines – removes all the pattern from the chart except the last ‘Splines In Memory’
    • Splines In Memory – leave the last pattern on the chart
    • Font Size – font size of the splines
    • Font Y Coordinate Accumulation – Y offset of the text relative to the anchor point
    • Font X Coordinate Accumulation – X offset of the text relative to the anchor point
      Enable/disable searching for different types of parabolas:

  • Searching PARABOLIC(a) – search for parabolas with the vertex at the distant candle and with a continuation as a horizontal line

    • Searching PARABOLIC(b) – search for parabolas with the vertex at the distant candle
    • Searching PARABOLIC(с) – search for parabolas with the vertex at the near candle
      Parameters of a single parabola with the name “Par. Name”:

  • “Par. Name” Label – name of the pattern

    • “Par. Name” Label Color – color of the name text
    • “Par. Name” Main Spline Color – color of the main spline
    • “Par. Name” Help Spline Color – color of the auxiliary spline
    • “Par. Name” Help Dot Color – color of the auxiliary dashed line
    • “Par. Name” Help Spline Y – the offset of the auxiliary spline from the top in points
    • “Par. Name” Min Points Size – the minimum value of the spline on the Y axis in points
    • “Par. Name” Max Points Size – the maximum value of the spline on the Y axis in points
    • “Par. Name” Min Bougie Size – the minimum value of the spline on the X axis in candles
    • “Par. Name” Max Bougie Size – the maximum value of the spline on the X axis in candles
    • “Par. Name” Exponent – the highest power of the initial parabola (2 – standard parabola)
    • “Par. Name” Relative Lengthening Left – extend the initial spline to the left (1.0 – extend by 2 times)
    • “Par. Name” Relative Lengthening Right – extend the initial spline to the right (1.0 – extend by 2 times)
    • “Par. Name” Average Deviation Of Peaks – relative average deviation of the peaks (High and Low)
    • “Par. Name” Maximum Deviation Of Peaks – relative maximum deviation of the peaks (High and Low)
    • “Par. Name” Average Deviation Of Medians – relative maximum deviation of medians
