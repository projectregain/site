Утилита Trade Panel – это профессиональная торговая панель для работы с ордерами и позициями в один клик. Предназначена для комфортной и быстрой торговли на финансовых рынках.
Оснащена функциями открытия, закрытия и сопровождения ордеров и позиций, так же различными информационными и оповещающими функциями.

Эта утилита не работает в тестере стратегий, бесплатная демо-версия здесь (работает только на CADCHF). Версия для MetaTrader 5. Версия Lite.

###

Основные особенности и возможности утилиты:

  • Три метода расчета лота для открытия ордера

    • Установка виртуальных ордеров.
    • Переворот позиций (Reverse).
    • Четыре вида трейлинг стопа (Trailing stop), в том числе трейлинг стоп по индикатору Trend Scanner.
    • Автоматическая установка stop loss в уровень безубыток (Breakeven).
    • Автоматическое и ручное частичное закрытие позиций.
    • Оповещение на касание трендовой линии.

    • Автоматический и ручной скриншот.

      Основные параметры:

    • Trade Panel – включение / отключение панели Trade.

    • Mode lot – метод расчета лота.
    • Lot (Fixed Lot) – значение лота в панели Trade и Line trade при методе Fixed Lot.
    • % Risk SL – значение процента в панели Trade и Line trade при методе % SL.
    • % Risk Equity – значение процента в панели Trade и Line trade при методе % Equity.
    • Step lots (Fixed Lot) – шаг изменения лота в панели Trade и Line trade при методе Fixed Lot.
    • Step % (% SL, % Equity) – шаг изменения процента риска в панели Trade и Line trade при методе % SL и % Equity.
    • Ratio – значение Ratio в панели Trade и Line trade.
    • Price deviation – уровень максимального проскальзывания цены при открытии и закрытии позиции.
    • Buttons reverse – включение / отключение кнопок reverse.
    • Buttons pending – включение / отключение кнопок отложенных ордеров.
    • Label binding side – сторона привязки меток для открытия ордеров.
    • Y distance label – расстояние от стороны до меток.
    • Line Trade Panel – включение / отключение панели Line trade.
    • Distance virtual order – дистанция между ценой bid и линией открытия виртуального отложенных ордеров.
    • Button screenshot – включение / отключение кнопки screenshot в панели инструментов.
    • Screenshot open position – включение / отключение автоматического screenshot при открытии позиции.
    • Screenshot open order – включение / отключение автоматического screenshot при открытии ордера.
    • Screenshot close position – включение / отключение автоматического screenshot при закрытии позиции.
    • Format – формат screenshot GIF, PNG или BMP.
    • Width (x pixels) – размер сохраняемого изображения в пикселях по X.
    • Height (y pixels) – размер сохраняемого изображения в пикселях по Y.
    • Alert Panel – включение / отключение панели Alert.
    • Type line alert – тип линии (горизонтальная или трендовая) для алерта по цене.
    • Distance line alert – дистанция между ценой bid и линией алертов по цене.
    • Trailing Panel – включение / отключение панели Trailing.
    • Spread for Trailing stop – размер спреда на инструменте, необходим для работы трейлинг стопа по фракталам и по индикатору Trend Scanner.
    • Use closed bar (Trend Scanner) – используется значение индикатора: true – последнего завершенного бара, false – незавершенного бара.
    • Show – вид отображаемой информации: серверное время, прибыль в валюте депозита, прибыль в пунктах и выключено.
    • Breakeven Panel – включение / отключение панели Breakeven.
    • Partial close Panel – включение / отключение панели Partial close.
    • Sound buttons – звук нажатия кнопок, доступны на выбор два встроенных звука, так же можно уст ановить пользовательский.
    • File name (user) – пользовательский звук для кнопок, выбираем в Sound buttons = user (название файла звука, файл должен находиться в папке Sounds).
    • Sound success – звук успешного события, доступны на выбор два встроенных звука, так же можно установить пользовательский.
    • File name (user) – пользовательский звук для успешного события, выбираем в Sound success = user (название файла звука, файл должен находиться в папке Sounds).
    • Sound error – звук ошибки, доступны на выбор два встроенных звука, так же можно установить пользовательский.
    • File name (user) – пользовательский звук для ошибки, выбираем в Sound error = user.
    • File name sound alert – звук алертов (название файла звука, файл должен находиться в папке Sounds).
