This is a auto-trading program based on Martingale.

Proposed to stop trading when important data or central bank meetings would be published.

The recommended chart timeframe is M1 or лю5 of EURUSD.

The recommended minimum deposit is $1 000.

Running the ECN account guarantees small spread.

Recommended to select a small delay VPS to run.

When you test this EA, skip the days of important events.

### Inputs

  • WinPoints: the target profit per 0.01 lot, default is 30. If this parameter is set too small, the profit is reduced. On the contrary, earnings increase, but at the same time will increase the risk.

    • MinIncPoints: the smallest interval for opening additional orders, default is 30.
    • *MAGICNUM: * magic number.
    • *Risk: * risk value, 0-10.
    • *StartHour: *start hour (0-23), if you want to set EA work time interval in 24 hours, set the start hour by this input.
    • *EndHour: *end hour (0-23), end hour of EA running.
      For example, you want to let this EA work during 0:00-8:59 , you can set StartHour=0, EndHour=9. EA would start to open orders from 0:00. When the time is 9 clock, all opened orders would be closed.

    If you want to let the EA work for 24 hours, just set StartHour=EndHour=AnyIntegerValue.
