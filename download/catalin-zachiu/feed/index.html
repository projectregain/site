The EA buys when RSI is overbought and sells when RSI is oversold with a 100 Point Profit Target and a 1000 Point Stop . If the result from the first trade is positive , the EA will open the next order in reverse , buying when the RSI is oversold and viceversa untill it gets a loss . If it gets a loss it reverses order direction again .

The “HR” from the name comes from “High Risk” because if it gets, a loss it will recover it by increasing Lot size with the help of the implemented Recovery Factor which is set to 12 to recover the 1000 Points lost .

The EA uses two levels of recovery , after which it does not increase lot size any more to prevent further losses .

Timeframe : H1 ;

Minimum recomended Balance : 500 USD ;

Parameters :

Lots – for fixed Lot setting ;* TakeProfit, set to 100 Points ; *StoLoss ,set to 1000 Points ; TrailingStop in Points ;

RecoveryFactor set to 12 ; RSIPeriod = 14 ;* RSIUpperLevel* and RSILowerLevel *; *MagicNumber to differentiate from other EA’s.
