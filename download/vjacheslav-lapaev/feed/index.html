Basic statistics and variation series.

### Comment to the algorithm

Basic statistics and variation series are calculated once in the process of initialization. Algorithms for calculating the mean standard deviation and corresponding frequency shift (probabilities) are built directly in the indicator.

### Comment to trading

The indicator is designed for monitoring the most important dynamic statistics (mean standard deviation) and probability of occurrence of the fluctuation band of the given symbol’s prices.

### Inputs

Block I

  • financialInstrument — financial instrument the indicator performs its calculations for

    • theScaleFactor — scale ratio for displaying numbers (1, 10, 100, 1000).
      Block II

  • theNumberOfBarsToCalculateTF — number of candles on the current timeframe to calculate the indicator’s parameters;

    • theNumberOfBarsToCalculateH1 — number of candles on H1 to calculate the indicator’s parameters;
    • theNumberOfBarsToCalculateH4 — number of candles on H4 to calculate the indicator’s parameters;
    • theNumberOfBarsToCalculateD1 — number of candles on D1 to calculate the indicator’s parameters;
    • theNumberOfBarsToCalculateW1 — number of candles on W1 to calculate the indicator’s parameters;
    • theNumberOfBarsToCalculateMN1 — number of candles on MN1 to calculate the indicator’s parameters.
      Block III

  • font — font for all elements of the indicator

    • mainFontSize — font size for the main headings;
    • dataFontSize — font size for the data headings;
    • curFontSize — font size for the current data;
    • valFontSize — font size for the main statistics data;
    • varsFontSize — font size for the variation series data.
      Block IV

  • shift_Y_Total — total shift of all objects of the indicator along the Y axis;

    • shift_X_Total — total shift of all objects of the indicator along the X axis.
      Block V

  • shift_1R1C — shift of all elements in the 1st line.
    Block VI

  • shift_2R1C — shift of the 1st element in the 2nd line;

    • shift_2R2C — shift of the 2nd element in the 2nd line;
    • shift_2R3C — shift of the 3rd element in the 2nd line.
      Block VII

  • shift_3R1C — shift of the 1st element in the 3rd line;

    • shift_3R2C — shift of the 2nd element in the 3rd line;
    • shift_3R3C — shift of the 3rd element in the 3rd line;
    • shift_3R4C — shift of the 4th element in the 3rd line.
      Block VIII

  • shift_4RNC — shift of all elements in the 4th line;

    • shift_5RNC — shift of all elements in the 5th line.
      Block IX

  • shift_6RNC — shift of all elements in the 6th line;

    • shift_7RNC — shift of all elements in the 7th line;
    • shift_8RNC — shift of all elements in the 8th line;
    • shift_9RNC — shift of all elements in the 9th line;
    • shift_10RNC — shift of all elements in the 10th line;

    • shift_14RNC — shift of all elements in the 14th line;

    • shift_vs — distance between columns containing elements of the indicator.
