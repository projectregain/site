Gendhis GBPUSD scalper is a fully automated Expert Adviser that works by observing speed of price movements with Bollinger Bands filter. This EA uses pending orders in upper or lower Bollinger Bands to search for breakout price.

This EA is designed for GBPUSD only.

### Requirements

Not all brokers are suited to run this EA. Only a broker with a tight spread, tight stop level and order execution time of less than 100 milliseconds.

  1. 5 digit broker.

    1. It is recommended to use an ECN broker with a low spread and fast execution.
    2. The minimum leverage is 1:300.

    Parameters

  2. Fixed Lots: only used if UseMM (money management) is set to false.

    • Money Management: Set it True to use auto Lots Size.
    • Risk percentage: Risk setting in percentage of the balance, for 10.0 the risk is 10% of the balance.
    • Max Spread: Maximum spread can be allowed for EA to trade ( in pips )
    • Stop Loss: Stop loss from as many pips.
    • Start Trailing: Distance from an order price to start trailing (in pips).
    • Step Trailing: Distance from Current price to trailing (in pips).
    • Start Hour: Start Hour for open order
    • Stop Hour: Hour for ending open order
