    The Binary Sane MT5 indicator is designed for trading binary options. The indicator works on all currency pairs from M1 to M30. The additional timeframes in MT5 not present in MT4 allow to use the indicator not only on the options with the expiration time of 1, 5, 15, 30 minutes, but also on 2, 3, 4, 6, 10, 12, 20 minute options. The indicator displays the expected bar closure direction at every bar. The indicator has no parameters for adjusting the sensitivity. For ease of estimation of the probability characteristics of the indicator, the History input parameter is used. When the History=true mode is selected, the indicator displays the estimated price movement direction on the current bar and the indicator triggering history on all available bars, when the History=false mode is selected, only the prediction for the current bar is displayed.

    Input parameter:

    History – selection of the signal display mode with the triggering history or only the prediction for the current bar.
