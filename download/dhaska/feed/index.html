This indicator helps to draw support and resistance lines. Simply point your mouse to the level where you want to draw a line and press numpad button from 1 to 3. If you press 1 it means that you want to draw line for strong support and resistance. If you press 2 it means that you want to draw line for normal *support and resistance. If you press 3 it means that you want to draw line for *weak support and resistance. You can set up how these lines should look up in settings of the indicator. If you want to delete all levels, press 9 at numpad.

Settings also allow to set up drawing SUPPORT and RESISTANCE ZONES! You just define interval in pips in the settings and draw in same way like a SR lines.

### Settings

  • StrongZones – enable drawing zones instead of lines (default: false)
    • StrongZonesPips – interval for strong zone in pips
    • NormalZones – enable drawing zones instead of lines (default: false)
    • NormalZonesPips – interval for normal zone in pips
    • WeakZones – enable drawing zones instead of lines (default: false)
    • WeakZonesPips – interval for weak zone in pips
    • InBackground – show zones in the background (default: true)
    • StrongColor – color for the strong SR line
    • StrongZoneColor – color for the strong SR zone
    • StrongWidth – strong SR line width
    • StrongStyle – strong SR line style
    • NormalColor – color for the normal SR line
    • NormalZoneColor – color for the normal SR zone
    • NormalWidth – normal SR line width
    • NormalStyle – normal SR line style
    • WeakColor – color for the weak SR line
    • WeakZoneColor – color for the weak SR zone
    • WeakWidth – weak SR line width
    • WeakStyle – weak SR line style
