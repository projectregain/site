This utility copies deals from МТ4 to МТ4 terminal (both should be installed on one PC) and features the built-in risk management. It can copy from one account to several ones and vice versa. The built-in risk manager sets risks to protect your deposit. The copier can be useful both for those who simply wants to copy deals to multiple accounts and those who hires a trader but does not know how to control him/her. In the latter case, the copier imposes limitations by copying trader’s deals to accounts using the risk manager. You should specify only the operation mode when launching the product. The interactive panel appears on a symbol chart allowing you to manage the risk manager and copier.

### Inputs

  • master – source account.

    • slave – recipient account.

    Panel Settings

  • Timer – timer for limiting copying by time. There are two modes that can be set in the Mode field: 1 – start copying only at the time specified in the Time field daily, 2 – copy from Monday to Friday according to the Time fields: start time on Monday in the first one and end time on Friday in the second. If the option is unchecked, the timer is disabled.

      • Acceleration Profit Stop – Включить отключить ускорение достижения цели, чем больше сделок тем короче цель закрытия
      • Deposit load – Лимит загрузки депозита, при включенном параметре Open Position minlot открываться минимальным лотом, при отключенном, не открывать больше сделок.
