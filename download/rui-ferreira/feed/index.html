Telegram Signals Manager is a fully featured manager for your Telegram signal channel, which doesn’t require any VPS. The messages are fully customizable, and support emojis. It sends messages to your telegram channel with signal data, and updates about each signal.

These are the features of Telegram Signals Manager:

  • Send customizable signals to your channel.

    • Updates about SL/TP hits are automatically sent to your channel* (doesn’t require any VPS!).*
    • When TP1 hits, the EA will internally set StopLoss to entrypoint. When TP2 hits, the SL will be set to the TP1 level.
    • Send currentlysignals.
    • Send daily closed signals.
    • Send weekly closed signals.
    • You can customize your signals to use up to 3 Take Profit levels.
    • Update your Stop Loss levels after a signal has been sent.
    • Send a message to change Stop Loss to entrypoint.
    • Send a message to close a certain signal.

    Configuration

    Run *Telegram Signals Manager *only on one chart. The required configuration, is inside the Help tab of the EA interface. Change the chart symbol, to automatically fill the values, or use the interface.
