The EA trades in the fully automated mode and uses the breakeven function when price moves half of the distance towards the take profit.

The Relax Time EA trades on H4, which provides a very accurate testing on history.

Note: Set the correct GMT offset for the proper operation of the EA.

  2. Make a simple calculation:

    Terminal time minus GMT = your broker’s GMT

    In my example: 09:xx – 06:xx = 3

    My broker’s GMT = 3
