Этот коннектор позволяет совершать сделки из терминала MT5 на платформе брокера BinaryCom. Для соединения с брокером потребуется ваш ключ ID который можно получить у брокера. Для связи советника с брокером необходимо разрешить отправку WebRequest на указанный сайт. Коннектор позволяет ускорить работу с платформой и анализировать графики непосредственно на платформе MT5.

### Входные параметры:

  1. Site name– Сайт, необходимый для работы коннектора

    1. *Account Type *– Тип вашего аккаунта (валюта счета)
    2. Symbol name– Валюта торговли
    3. Broker ID– Ваш персональный id у брокера, для совершения сделок. Этот id привязан к вашему счету
    4. Name of file – Имя файла, для записи ваших сделок (оставить пустым, если не нужна эта функция )
    5. *Add Time- *время отображения сигналов после их завершения в таблице сделок (секунд / тиков)
    6. Payout– минимальная выплата, необходимая для открытия сделки
    7. Step of time– Шаг времени, для изменения на панели коннектора
    8. Step of stake– Шаг ставки, для изменения на панели коннектора
    9. *Synchronization *– Пытаться отслеживать время сделки параллельно её открытию/закрытию (особенности MT)
    10. Check Balance In– время, для синхронизации баланса (в минутах)
    11. Up Arrow Name *- *сигнал для автоторговли, для открытия позиции на повышение
    12. *Down Arrow Name *– сигнал для автоторговли, для открытия позиции на понижение
    13. Moneymanagment Type – тип управления капиталом для автоторговли
    14. Max steps – максимальное количество ступеней при алготорговле
    15. X Rate – множитель для управления капиталом
    16. Stop on last step – останавливать торговлю при достижении максимального шага / колена
    17. Take profit – величина тэйк профита для алготорговли
    18. Stop loss – величина стоп лоса для алготорговли
    19. Trailing Stop – трэйлинг стоп для алготорговли

    Особенности задержки : при работе через данный коннектор, обратите внимание на наличие задержки. Как правило задержка составляет около 1 сек / 1 тик. На это влияет несколько факторов: загруженность брокера, приход тиков по валюте, загруженность сайта, ваше местоположение.

    Особенности отправки запросов: при отправке запроса, из-за особенностей MQL, таймеры отсчета сделок останавливаются. Учтите этот момент, при работе с статистикой робота. (В 3 версии есть возможность синхронизировать сделки)
